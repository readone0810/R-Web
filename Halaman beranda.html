Prakata

Selamat datang di web R-Math para pembaca yang budiman, perkenalkan nama saya Ridwan Kusnadi yang merupakan admin dari web ini dan saya adalah mahasiswa aktiif UNIVERSITAS PASUNDAN FAKULTAS KEGURUAN DAN ILMU PENDIDIKAN PROGRAM STUDI PENDIDIKAN MATEMATIKA. Tujuan

tujuan saya membuat web ini adalah sebagai media pembelajaran bagi teman - teman untuk mempelajari matematika dari dasar sampai dari tahap lanjutan. walaupun web ini didesain untuk matematika dasar dari SD sampai dengan SMA dan masih banyak yang harus dibenahi, tetapi saya akan terus menyempurnakan web ini. tujuan saya adalah menciptakan media pembelajaran berbasis web yang multi tools, interaktif dan komunikatif. seperti dalam web ini terdapat tools kalkulator grafik yang dapat membuat grafik dan dapat meningkatkan ketertarikan dalam belajar dan memahami materi matematika.

Harapan

Harapan saya web ini dapat terus berkembang dan dapat diakses oleh banyak orang. sehingga web ini dapat bermanfaat untuk oranglain dan dijadikan media untuk belajar matematika. harapan saya juga beriring dengan berjalannya waktu saya dapat lebih mengembangkan, memajukan dan menyempurnakann web ini dengan tools dan fitur yang lebih interaktif dan komunikatif serta menghadirkan AI didalam nya untuk menambah efisieni, efektivitas dan produktivitas web ini sebagai media pembelajaran matematika. sehingga web ini akan sangat relevan dalam pembelajaran abad 21 yang berbasis dengan teknologi.
